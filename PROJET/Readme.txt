Description des fichiers :
- Le fichier expr.l contient l'analyse lexicale lex
- Le fichier expr.y contient la grammaire yacc
- Les fichiers quad.h et quad.c contiennent les fonctions associees aux quads
- Les fichiers symbolTable.h et symbolTable.c contiennent les fonctions
associees a la table des symboles


Commandes utiles :
<make>       : pour compiler
<make run>   : pour lancer le programme
<make clean> : pour nettoyer
